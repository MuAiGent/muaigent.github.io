# MuAiGent 个人作品集网站

## 项目概述

这是一个面向AI产品经理的专业作品集展示网站，采用现代化UI设计与交互体验，专注于展示AI、游戏、XR领域的项目经验与技能。网站采用响应式设计，适配从移动设备到桌面端的各类设备，并针对不同屏幕尺寸优化了内容布局与交互方式。

![网站预览](assets/images/header-banner.webp)

## 核心功能

### 1. 作品展示系统
- **多模式查看**: 支持缩略图浏览和详情展开两种查看模式
- **动态内容加载**: 基于JavaScript动态渲染作品内容，便于维护与更新
- **分类标签**: 通过标签系统快速识别项目类型与相关技能
- **展开/收起机制**: 高级交互设计，允许用户查看更多项目细节而不影响整体浏览体验

### 2. 响应式布局
- **全设备适配**: 从移动设备到大型显示器的无缝适配
- **自适应导航**: 根据设备类型提供最佳导航体验
- **弹性内容区**: 内容区域根据屏幕尺寸动态调整，确保最佳阅读体验
- **触控优化**: 为触摸设备优化的交互元素尺寸与位置

### 3. 专业信息展示
- **分区化设计**: 清晰划分个人简介、技能展示、项目经验等不同区块
- **层级信息结构**: 采用信息分层展示，便于访问者快速获取关键信息
- **重点突出**: 视觉设计上强调核心能力与重要项目
- **用户引导**: 内容组织引导用户自然浏览完整作品集

## 技术实现

### 前端技术栈
- **HTML5**: 语义化标签结构，提升SEO与可访问性
- **CSS3**: 
  - Flexbox/Grid布局系统
  - CSS变量实现主题管理
  - 媒体查询驱动的响应式设计
  - 高效CSS选择器设计
- **JavaScript (ES6+)**: 
  - 模块化组织
  - 事件委托优化
  - DOM操作最小化
  - 节流与防抖优化

### 性能优化策略
- **资源优化**: 
  - WebP格式图片替换传统格式，减少40%+加载体积
  - 图像压缩与尺寸优化
  - 渐进式图像加载
- **代码优化**:
  - CSS选择器优化
  - JavaScript执行效率优化
  - 减少重绘与回流操作
- **加载策略**:
  - 首屏内容优先加载
  - 非关键资源延迟加载
  - 图片懒加载实现

### 交互设计创新
- **智能滚动记忆**: 展开/收起内容时记住滚动位置，提升用户体验
- **弹性动画系统**: 基于CSS transitions的平滑过渡动画
- **多设备交互适配**: 针对鼠标/触摸设备提供不同交互反馈
- **备用操作方案**: 为关键功能提供多种操作路径，提高可访问性

## 开发与部署指南

### 本地开发环境设置
1. 克隆代码库: `git clone https://github.com/MuAiGent/muaigent.github.io.git`
2. 打开项目: 使用VS Code或其他编辑器打开项目目录
3. 启动本地服务器: 
   - 使用VS Code Live Server插件
   - 或Python: `python -m http.server 8000`
   - 或Node.js: `npx serve`
4. 访问 `http://localhost:8000` 或相应端口查看效果

### 内容更新流程
1. **作品更新**: 修改`index.html`中的`portfolioData`对象
   ```javascript
   const portfolioData = {
     'project-key': {
       title: '项目标题',
       image: 'assets/images/项目主图.webp',
       tags: ['标签1', '标签2'],
       brief: '项目简介...',
       expandImages: [
         'assets/images/详情图1.webp',
         'assets/images/详情图2.webp'
       ]
     },
     // 更多项目...
   };
   ```
2. **样式调整**: 修改CSS变量实现全局风格更新
3. **图片资源**: 建议使用WebP格式，优化尺寸至适合网络传输

### 部署方法
- **GitHub Pages**: 推送到GitHub仓库自动部署
- **Netlify/Vercel**: 连接至代码库实现自动部署
- **传统托管**: 上传所有文件至支持静态网站的服务器

## 特色优化

### 用户体验优化
1. **智能按钮定位**: 
   - 详情按钮总是可见，解决移动设备上按钮被遮挡问题
   - 右上角额外收起按钮，确保用户随时可以退出详情视图
2. **滚动位置记忆**:
   - 展开内容后记录原始位置
   - 收起时自动恢复到原始位置，避免用户迷失
3. **平滑过渡动画**:
   - 所有状态变化均有平滑过渡
   - 动画时长经优化，平衡流畅感与响应速度

### 响应式设计优化
1. **关键断点优化**:
   - 桌面端 (>992px): 双栏布局，侧边栏固定
   - 平板端 (768px-992px): 双栏布局，栏宽优化
   - 移动端 (<768px): 单栏布局，垂直组织内容
2. **元素适应性调整**:
   - 按钮大小随屏幕变化
   - 文字大小自适应
   - 内容边距动态调整

### 性能优化成效
1. **加载速度提升**:
   - WebP图片格式减少约45%加载时间
   - CSS选择器优化减少渲染时间
2. **交互响应优化**:
   - 滚动性能优化，确保60fps流畅度
   - 点击反馈延迟<100ms
