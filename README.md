## 项目概述

这是一个面向AI产品经理的专业作品集展示网站，采用现代化UI设计与交互体验，专注于展示AI、游戏、XR领域的项目经验与技能。网站采用响应式设计，适配从移动设备到桌面端的各类设备，并针对不同屏幕尺寸优化了内容布局与交互方式。

![网站预览](assets/images/header-banner.webp)

### 核心价值主张

- **专业形象塑造**：为AI从业者提供专业的在线展示平台，突出技术深度与行业洞察
- **多维度能力展示**：立体呈现技术、产品、设计、管理等全方位能力
- **便捷内容管理**：基于结构化数据模型的内容更新机制，无需深入了解代码实现
- **优质用户体验**：精心设计的交互细节，确保信息获取的流畅性与直观性

## 核心功能

### 1. 作品展示系统
- **多模式查看**: 支持缩略图浏览和详情展开两种查看模式
- **动态内容加载**: 基于JavaScript动态渲染作品内容，便于维护与更新
- **分类标签**: 通过标签系统快速识别项目类型与相关技能
- **展开/收起机制**: 高级交互设计，允许用户查看更多项目细节而不影响整体浏览体验
- **双向操作保障**: 为每个关键操作提供多个交互入口，增强用户体验安全感

### 2. 响应式布局
- **全设备适配**: 从移动设备到大型显示器的无缝适配
- **自适应导航**: 根据设备类型提供最佳导航体验
- **弹性内容区**: 内容区域根据屏幕尺寸动态调整，确保最佳阅读体验
- **触控优化**: 为触摸设备优化的交互元素尺寸与位置
- **断点精细控制**: 在关键屏幕尺寸处实现精确的布局转换

### 3. 专业信息展示
- **分区化设计**: 清晰划分个人简介、技能展示、项目经验等不同区块
- **层级信息结构**: 采用信息分层展示，便于访问者快速获取关键信息
- **重点突出**: 视觉设计上强调核心能力与重要项目
- **用户引导**: 内容组织引导用户自然浏览完整作品集
- **数据驱动展示**: 基于真实项目数据的可视化呈现

## 项目亮点示例

### 作品展示区交互优化

以下是针对移动端环境下"查看详情"功能的交互优化示例：

```javascript
// 解决方案：增加多入口关闭机制
function toggleDetails(btn) {
    const item = btn.closest('.portfolio-item');
    const overlay = item.querySelector('.portfolio-overlay');
    
    if (overlay.classList.contains('active')) {
        // 收起详情，恢复滚动位置
        overlay.classList.remove('active');
        btn.innerHTML = '▲ 查看详情';
        // 智能恢复滚动位置
        if (item.dataset.scrollY) {
            window.scrollTo({
                top: parseInt(item.dataset.scrollY),
                behavior: 'smooth'
            });
        }
    } else {
        // 记录当前滚动位置，展开详情
        item.dataset.scrollY = window.scrollY;
        overlay.classList.add('active');
        btn.innerHTML = '▲ 收起';
    }
}
```

**技术亮点**：
- 智能滚动位置记忆与恢复
- 多入口操作设计
- 平滑过渡动画
- 明确的状态反馈

1. 克隆代码库: `git clone https://github.com/MuAiGent/muaigent.github.io.git`

**智能滚动位置记忆机制**:
```javascript
// 展开/收起时智能记忆与恢复滚动位置
function toggleExpand(button, key) {
    const portfolioItem = button.closest('.portfolio-item');
    const expandSection = portfolioItem.querySelector('.portfolio-expand-section');
    
    if (expandSection.classList.contains('active')) {
        // 收起时，先记住当前展开项的顶部位置
        const itemTop = portfolioItem.offsetTop;
        
        // 收起内容
        expandSection.classList.remove('active');
        button.textContent = '▼ 展开';
        expandSection.style.maxHeight = '0';
        
        // 平滑滚动回原位置
        window.scrollTo({
            top: itemTop - 20, // 减去偏移量优化视觉效果
            behavior: 'smooth'
        });
    } else {
        // 展开前记住当前位置
        portfolioItem.dataset.scrollPosition = window.scrollY;
        
        // 展开内容
        expandSection.classList.add('active');
        button.textContent = '▲ 收起';
        expandSection.style.maxHeight = (expandGrid.scrollHeight + 60) + 'px';
    }
}
```
